This folder contains following files ,
1. run_ques (which is used to run a simulation)
2. run.do file (provides the hierarchies and flags)
3. wrapper.do file (Wavefrom File containing input and Ouput ports of all Modules)
4. run_reg.sh file (Regeression Report Script)
5. run_script.sh file (is the regression script for all the testcase mentioned in VTP)
6. ".f" files (are RTL filelist used for compilation)
7. coverage_merge.sh file (used for merging Coverage and taking report)
8. ./run_ques $TESTNAME <symbol> <subframe> <DFE> <IFFT-FFT> <SCH> <long/short Prach>
